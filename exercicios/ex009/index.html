Listas ordenadas

Para definir uma lista ordenada basta usar a tag <ol>

Já para definir os itens da lista use a tag <li>. Além do mais,
podemos definir o início do índice da lista basta basta colocar o seguinte parâmetro: <ol type "" start"">.

  1. Acordar
  2. Escovar os dentes
  3. Tomar café
  4. Alongar
  5. Ir para a academia
  6. Tomar banho
  7. Ir trabahar

Listas não ordenada

Para definir uma lista não oedenada basta usar a tag <ul>.Além do mais, a tag possui os seguintes parâmetros : "circle,squar e disc";

  • café
